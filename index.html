Codepedia — свободная энциклопедия о программировании, которую может редактировать каждый. Мы собираем знания о языках, алгоритмах, концепциях и инструментах, чтобы сделать их доступными для всех. Проект основан на принципах Wikipedia: нейтральность, проверяемость, свобода знаний.

🎲 Случайная статья: Объектно-ориентированное программирование — парадигма, в которой программа строится как набор взаимодействующих объектов.

📖 Языки программирования

Python

Высокоуровневый язык с динамической типизацией. Прост для изучения, огромная экосистема библиотек для Data Science и ML.

#динамический #интерпретируемый

C++

Мощный компилируемый язык для системного программирования. Даёт полный контроль над памятью и железом.

#компилируемый #статический

Java

Объектно-ориентированный язык, работающий на JVM. "Write once, run anywhere" — кроссплатформенность.

#jvm #объектно-ориентированный

JavaScript

Язык для веб-разработки. Работает в браузере и на сервере (Node.js). Самый популярный язык в мире.

#веб #асинхронный

Go (Golang)

Язык от Google. Простой синтаксис, встроенная конкурентность (горутины), быстрая компиляция.

#системный #конкурентный

Rust

Безопасный системный язык без сборщика мусора. Проверка заимствований на этапе компиляции.

#безопасный #системный

🧮 Алгоритмы и структуры данных

Алгоритмы сортировки

Quicksort, Mergesort, Heapsort, Bubblesort — сравнение сложности и реализация на Python и C++.

#сортировка #o(n-log-n)

Алгоритмы поиска

Бинарный поиск, линейный поиск, хеш-таблицы. Как быстро найти элемент в массиве.

#поиск #o(log-n)

Алгоритмы на графах

Dijkstra, BFS, DFS, поиск кратчайшего пути. Обход графов и работа с сетями.

#графы #пути

🧠 Ключевые концепции

ООП (Объектно-ориентированное программирование)

Наследование, полиморфизм, инкапсуляция, абстракция. Основные принципы ООП.

#парадигма #объекты

Функциональное программирование

Чистые функции, неизменяемость данных, функции высшего порядка, map/filter/reduce.

#парадигма #функции

Указатели и ссылки

Работа с памятью в C/C++. Указатели, ссылки, умные указатели (unique_ptr, shared_ptr).

#память #c-cpp

🛠️ Инструменты разработчика

Git

Система контроля версий. Коммиты, ветки, merge, rebase, работа с удалёнными репозиториями.

#vcs #github

Docker

Контейнеризация приложений. Dockerfile, образы, контейнеры, docker-compose.

#devops #контейнеры

Visual Studio Code

Популярный редактор кода от Microsoft. Расширения, отладка, интеграция с Git.

#редактор #ide
