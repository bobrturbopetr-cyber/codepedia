Политика конфиденциальности

Codepedia уважает вашу приватность. Эта страница объясняет, как мы обрабатываем данные.

1. Какие данные мы собираем

Сайт использует Яндекс.Рекламу, которая может собирать обезличенные данные о посетителях (IP-адрес, тип браузера, страницы просмотра) для показа релевантной рекламы.

2. Как используются данные

Данные используются исключительно для:

  • Показа персонализированной рекламы
  • Анализа посещаемости сайта
  • Улучшения работы сайта

3. Файлы cookie

Сайт может использовать файлы cookie для хранения настроек пользователя и работы рекламных систем.

4. Сторонние сервисы

На сайте используется Яндекс.Реклама. Данные обрабатываются согласно политике конфиденциальности Яндекса.

5. Ваши права

Вы можете отключить персонализированную рекламу в настройках вашего аккаунта Яндекса или использовать браузер с режимом "Инкогнито".

6. Контакты

По вопросам конфиденциальности: GitHub Issues
