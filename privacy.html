Политика конфиденциальности

Codepedia (далее — "Сайт") уважает вашу приватность. Эта страница объясняет, как мы обрабатываем данные.

1. Какие данные мы собираем

Сайт использует Яндекс.Рекламу (РСЯ), которая может собирать обезличенные данные о посетителях:

  • IP-адрес (в обезличенном виде)
  • Тип браузера и устройства
  • Просматриваемые страницы
  • Время посещения

2. Как используются данные

Данные используются исключительно для:

  • Показа релевантной рекламы
  • Анализа посещаемости сайта
  • Улучшения работы сайта

3. Файлы cookie

Сайт и рекламная сеть Яндекса используют файлы cookie для хранения настроек пользователя и работы рекламных систем. Вы можете отключить cookie в настройках браузера.

4. Сторонние сервисы

На сайте используется Яндекс.Реклама. Данные обрабатываются согласно политике конфиденциальности Яндекса.

5. Ваши права

Вы можете:

  • Отключить персонализированную рекламу в настройках Яндекса
  • Использовать браузер в режиме "Инкогнито"
  • Очистить файлы cookie в любое время

6. Контакты

По вопросам конфиденциальности: GitHub Issues
