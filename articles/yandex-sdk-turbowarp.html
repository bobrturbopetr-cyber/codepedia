📖 Обзор

Yandex Games SDK Extension — это расширение для TurboWarp (и совместимых сред Scratch), которое реализует полный набор функций Яндекс.Игр SDK версии 1.19.

Расширение позволяет разработчикам, использующим визуальное программирование, публиковать свои проекты на платформе Яндекс.Игр с полной поддержкой всех необходимых функций.

✅ Возможности расширения:
  • Инициализация SDK
  • Полноэкранная реклама
  • Видеореклама с наградой
  • Сохранение прогресса игрока
  • Геймплей-метрики (аналитика)
  • Автоопределение языка и региона
  • Определение типа устройства
  • Оценка игры (rate)
  • Управление звуком при сворачивании
  • События паузы/возобновления

⚙️ Установка

Способ 1: Через URL (TurboWarp)

  1. Откройте TurboWarp (или стандартный Scratch с турбо-режимом)
  2. Перейдите в меню "Расширения"
  3. Выберите "Загрузить расширение по URL"
  4. Вставьте ссылку на JS-файл расширения
  5. Нажмите "Загрузить"

Способ 2: Локальная установка

  1. Скачайте JS-файл расширения
  2. В TurboWarp: Файл → Загрузить расширение из файла
  3. Выберите скачанный файл

Способ 3: Встраивание в HTML (для публикации)

<script src="https://yandex.ru/games/sdk/v1/sdk.js"></script>
<script src="yandex-sdk-extension.js"></script>

🚀 Инициализация

Для корректной работы расширения необходимо последовательно выполнить несколько шагов:

Когда зеленый флаг нажат
    init sdk
    ждать пока sdk ready
    load player data
    game ready call last
💡 Важно: Блок game ready call last должен вызываться последним из всех инициализационных блоков. Он сообщает платформе Яндекс.Игр, что игра полностью загружена и готова к работе.

📺 Реклама

Расширение поддерживает два типа рекламы: полноэкранную и видеорекламу с наградой.

Полноэкранная реклама

Когда зеленый флаг нажат
    show fullscreen ad
    когда ad closed
        ...

Реклама с наградой

Когда зеленый флаг нажат
    show rewarded ad
    когда rewarded ad watched
        если give reward, то
            выдать награду игроку
⚠️ Важно по документации Яндекс.Игр:
  • Рекламу нельзя показывать без инициализации SDK
  • Рекламу нельзя показывать чаще 1 раза в 30 секунд
  • При показе рекламы геймплей должен быть остановлен через stop gameplay
  • После закрытия рекламы геймплей нужно возобновить через start gameplay

💾 Сохранения (п.1.9 документации)

Яндекс.Игры предоставляют облачное хранилище для сохранения прогресса игрока. Данные привязаны к аккаунту пользователя и синхронизируются между устройствами.

// Сохраняем прогресс
save player data {"level": 5, "score": 12450, "items": ["sword", "shield"]}

// Загружаем и читаем
load player data
player data  // возвращает {"level":5,"score":12450,...}
💡 Совет: Сохраняйте данные в формате JSON. Это позволяет хранить сложные структуры: уровни, инвентарь, настройки.

🎮 Геймплей-метрики (п.1.19.3)

Метрики геймплея помогают платформе анализировать вовлечённость игроков. Яндекс использует эти данные для ранжирования игр в каталоге.

Когда зеленый флаг нажат
    start gameplay      // начали отсчёт времени игры
    
    // ... игровой процесс ...
    
    stop gameplay       // остановили отсчёт (например, при паузе)
✅ Правила использования:
  • start gameplay — когда игрок активно взаимодействует с игрой
  • stop gameplay — при паузе, открытии меню, показе рекламы
  • Не вызывайте эти блоки без необходимости
