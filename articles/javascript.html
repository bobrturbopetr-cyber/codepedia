JavaScript

📌 Кратко: Язык программирования для веб-разработки. Работает в браузере и на сервере (Node.js). Создан за 10 дней в 1995 году.

JavaScript — интерпретируемый язык с прототипным наследованием. Несмотря на название, не связан с Java. Сегодня — самый популярный язык в веб-разработке.

✨ Особенности

  • Асинхронность — Promises, async/await
  • Динамическая типизация
  • Прототипное наследование
  • Событийно-ориентированный
  • Первое место по количеству библиотек (npm)

💻 Пример кода

// Hello World
console.log("Hello, Codepedia!");

// Функция
const greet = (name) => `Hello, ${name}!`;

// Класс (ES6+)
class Developer {
    constructor(name, language) {
        this.name = name;
        this.language = language;
    }
    
    code() {
        console.log(`${this.name} пишет на ${this.language}`);
    }
}

// Асинхронность
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Ошибка:', error);
    }
}

// Массивы и методы
const numbers = [1, 2, 3, 4, 5];
const squares = numbers.map(n => n * n);
const evens = numbers.filter(n => n % 2 === 0);

// Использование
const dev = new Developer("Анна", "JavaScript");
dev.code();
