🔍 Линейный поиск (Linear Search)

Самый простой алгоритм. Последовательно перебирает все элементы. Работает на любом массиве.

def linear_search(arr, target):
    """Возвращает индекс элемента или -1"""
    for i, val in enumerate(arr):
        if val == target:
            return i
    return -1

# Пример
arr = [5, 2, 8, 1, 9]
print(linear_search(arr, 8))  # Вывод: 2
// C++ версия
int linear_search(const std::vector& arr, int target) {
    for (size_t i = 0; i < arr.size(); i++) {
        if (arr[i] == target) return i;
    }
    return -1;
}

🎯 Бинарный поиск (Binary Search)

Работает только на отсортированном массиве. Делит массив пополам на каждом шаге. Сложность O(log n).

def binary_search(arr, target):
    """Бинарный поиск на отсортированном массиве"""
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = (left + right) // 2
        
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1

# Пример
arr = [1, 2, 5, 8, 9, 12, 15]
print(binary_search(arr, 8))  # Вывод: 3

🗂️ Хеш-таблицы (Hash Tables)

Поиск за O(1) в среднем. Использует хеш-функцию для прямого доступа к элементу.

# Python словарь (хеш-таблица)
lookup = {
    "apple": 1,
    "banana": 2,
    "cherry": 3
}

# Поиск за O(1)
print(lookup.get("banana"))  # Вывод: 2

# Проверка наличия
if "apple" in lookup:
    print("Яблоко найдено!")
