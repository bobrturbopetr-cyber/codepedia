Rust

📌 Кратко: Безопасный системный язык программирования, созданный Mozilla. Без сборщика мусора, но с проверкой заимствований.

Rust — язык, который даёт контроль C++ без утечек памяти и гонок данных. 8 лет подряд признаётся самым любимым языком программирования в опросах Stack Overflow.

✨ Особенности

  • Владельческая модель памяти — ownership (без сборщика мусора)
  • Заимствования и времена жизни — lifetimes
  • Нулевая стоимость абстракций — как в C++
  • Без segfault'ов — гарантия безопасности памяти
  • Cargo — лучший менеджер пакетов

💻 Пример кода

// Hello World
fn main() {
    println!("Hello, Codepedia!");
    
    // Владение (ownership)
    let s1 = String::from("hello");
    let s2 = s1; // s1 больше не действителен
    
    // Заимствование (borrowing)
    let len = calculate_length(&s2);
    println!("Длина '{}': {}", s2, len);
    
    // Структура
    struct Developer {
        name: String,
        language: String,
    }
    
    impl Developer {
        fn code(&self) {
            println!("{} пишет на {}", self.name, self.language);
        }
    }
    
    let dev = Developer {
        name: String::from("Анна"),
        language: String::from("Rust"),
    };
    dev.code();
    
    // Option и match
    fn divide(a: i32, b: i32) -> Option {
        if b == 0 { None } else { Some(a / b) }
    }
    
    match divide(10, 2) {
        Some(result) => println!("10 / 2 = {}", result),
        None => println!("Деление на ноль!"),
    }
}

fn calculate_length(s: &String) -> usize {
    s.len()
}
Реклама

🎯 Где используется Rust

  • Системное программирование — операционные системы (Redox OS)
  • Веб-движки — Servo (Mozilla)
  • Инструменты разработки — ripgrep, alacritty, fd
  • Блокчейн — Solana, Polkadot, Near
  • Встраиваемые системы — безопасные драйверы
  • Браузеры — компоненты Firefox на Rust
💡 Почему Rust? — Компилятор проверяет память во время компиляции, поэтому нет segfault'ов и гонок данных.
