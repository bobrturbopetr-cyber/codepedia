🚀 Быстрая сортировка (Quicksort)

Самый популярный алгоритм на практике. Использует стратегию "разделяй и властвуй".

def quicksort(arr):
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quicksort(left) + middle + quicksort(right)

# Пример
print(quicksort([3, 6, 8, 10, 1, 2, 1]))  # [1, 1, 2, 3, 6, 8, 10]
// C++ версия
template<typename T>
void quicksort(std::vector<T>& arr, int left, int right) {
    if (left >= right) return;
    
    T pivot = arr[(left + right) / 2];
    int i = left, j = right;
    
    while (i <= j) {
        while (arr[i] < pivot) i++;
        while (arr[j] > pivot) j--;
        if (i <= j) {
            std::swap(arr[i], arr[j]);
            i++;
            j--;
        }
    }
    
    quicksort(arr, left, j);
    quicksort(arr, i, right);
}

🔗 Сортировка слиянием (Mergesort)

Гарантирует O(n log n) даже в худшем случае. Стабильная и предсказуемая.

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    
    result.extend(left[i:])
    result.extend(right[j:])
    return result

🐌 Пузырьковая сортировка (Bubblesort)

Простая, но медленная. Используется только для обучения.

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

🎯 Что выбрать в реальном проекте?

✅ Готовые реализации (всегда предпочтительнее):

• Python: list.sort() (Timsort)
• Java: Arrays.sort() (Dual-Pivot Quicksort)
• C++: std::sort (Introsort)
• JavaScript: array.sort() (Timsort)

🎯 Особые случаи:
• Почти отсортированные данные → Insertion Sort
• Огромные данные (не влезают в RAM) → External Merge Sort
• Маленький диапазон значений → Counting Sort
