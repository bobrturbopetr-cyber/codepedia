Python — интерпретируемый язык, который славится своей читаемостью и простотой. Сегодня это один из самых популярных языков в мире, особенно в области data science и машинного обучения.

✨ Особенности

  • Динамическая типизация — типы определяются во время выполнения
  • Автоматическая сборка мусора — не нужен ручной контроль памяти
  • "Батарейки в комплекте" — огромная стандартная библиотека
  • Кроссплатформенность — работает на Windows, Linux, macOS
  • Простой синтаксис — код читается почти как английский текст

💻 Пример кода

# Hello World
print("Hello, Codepedia!")

# Функция с аннотациями типов
def greet(name: str) -> str:
    return f"Hello, {name}!"

# Класс
class Developer:
    def __init__(self, name, language):
        self.name = name
        self.language = language
    
    def code(self):
        print(f"{self.name} пишет на {self.language}")

# List comprehension
squares = [x**2 for x in range(10)]

# Декоратор
def timer(func):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"Время: {time.time() - start:.4f}с")
        return result
    return wrapper

@timer
def slow_function():
    sum(range(1000000))

# Использование
dev = Developer("Анна", "Python")
dev.code()
slow_function()
