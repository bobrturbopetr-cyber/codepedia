Go (Golang)

📌 Кратко: Язык программирования, разработанный в Google в 2007 году. Простой, быстрый, с встроенной конкурентностью.

Go — компилируемый язык со статической типизацией. Создан для решения проблем масштабируемости и сложности C++. Разработчики: Роб Пайк, Кен Томпсон (создатель Unix) и Роберт Гризмер.

✨ Особенности

  • Горутины — легковесные потоки (десятки тысяч одновременно)
  • Быстрая компиляция — секунды для больших проектов
  • Сборщик мусора — с низкой задержкой
  • Встроенный форматтер — gofmt (единый стиль кода)
  • Статическая типизация с выведением типов

💻 Пример кода

package main

import (
    "fmt"
    "time"
)

// Функция
func greet(name string) string {
    return fmt.Sprintf("Hello, %s!", name)
}

// Структура (вместо классов)
type Developer struct {
    Name     string
    Language string
}

// Метод структуры
func (d Developer) Code() {
    fmt.Printf("%s пишет на %s\n", d.Name, d.Language)
}

// Горутина
func printNumbers() {
    for i := 1; i <= 5; i++ {
        fmt.Printf("%d ", i)
        time.Sleep(100 * time.Millisecond)
    }
}

func main() {
    fmt.Println("Hello, Codepedia!")
    
    // Горутина
    go printNumbers()
    
    // Использование структуры
    dev := Developer{"Анна", "Go"}
    dev.Code()
    
    // Каналы (channels)
    ch := make(chan string)
    go func() {
        ch <- "Сообщение из горутины"
    }()
    
    msg := <-ch
    fmt.Println(msg)
    
    time.Sleep(500 * time.Millisecond)
}
Реклама

🎯 Где используется Go

  • Серверные приложения — Docker, Kubernetes, Prometheus
  • Микросервисы — высоконагруженные системы
  • CLI-инструменты — быстрые консольные утилиты
  • Облачные технологии — Terraform, InfluxDB, etcd
  • Сетевые сервисы — прокси, балансировщики
💡 Интересный факт: Go был создан, потому что разработчики Google ждали компиляцию C++ часами.
