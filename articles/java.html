Java — строго типизированный объектно-ориентированный язык. Программы компилируются в байт-код, который выполняется на Java Virtual Machine (JVM), что обеспечивает кроссплатформенность.

✨ Особенности

  • JVM — код работает на любой платформе с JVM
  • Сборщик мусора — автоматическое управление памятью
  • Многопоточность — встроенная поддержка потоков
  • Безопасность — строгая типизация, защита от утечек
  • Огромная экосистема — Spring, Hibernate, Maven, Gradle

💻 Пример кода

// Hello World
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Codepedia!");
    }
}

// Класс с ООП
public class Developer {
    private String name;
    private String language;
    
    public Developer(String name, String language) {
        this.name = name;
        this.language = language;
    }
    
    public void code() {
        System.out.println(name + " пишет на " + language);
    }
    
    public static void main(String[] args) {
        Developer dev = new Developer("Максим", "Java");
        dev.code();
        
        // Stream API (Java 8+)
        var languages = java.util.List.of("Java", "Python", "C++");
        languages.stream()
                 .filter(lang -> lang.startsWith("J"))
                 .forEach(System.out::println);
    }
}
